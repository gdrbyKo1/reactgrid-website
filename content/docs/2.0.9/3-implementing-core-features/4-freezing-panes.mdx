## Some about frozens

ReactGrid allows you to freeze rows to top and bottom, also column to left or right.

It's possible by adding dedicated props:
- `frozenLeftColumns`
- `frozenRightColumns`
- `frozenTopRows`
- `frozenBottomRows`

All of these properties include the number of columns or rows to freeze at the grid's edges.

2. In this example we will add one freeze on each grid edge

```tsx
return (
	<ReactGrid
		rows={state.rows}
		columns={state.columns}
		license={"non-commercial"}
		frozenLeftColumns={1}
		frozenRightColumns={1}
		frozenTopRows={1}
		frozenBottomRows={1}
	/>
);
```
