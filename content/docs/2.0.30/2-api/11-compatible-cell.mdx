`CompatibleCell` type extends [`Cell`](link) interface by adding `text` and `value` props.

| Property name     | Type     | Property description  |
| -----------       | -------- | -------------------------|
| `text`            | `string` | Text value of cell  |
| `value?`          | `number` | Alternative value |
