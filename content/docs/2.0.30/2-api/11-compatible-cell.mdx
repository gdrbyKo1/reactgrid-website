`CompatibleCell` type is a way to recognize field that has our generic `Cell` interface.

## Definition

```tsx
type Compatible<TCell extends Cell> = TCell & {
    text: string;
    value: number;
}
```

## Properties

| Property name     | Type     | Property description  |
| -----------       | -------- | -------------------------|
| `text`            | `string` | Text value of cell  |
| `value`           | `number` | Numeric value of cell, if there is no numeric value representation use `NaN` |
