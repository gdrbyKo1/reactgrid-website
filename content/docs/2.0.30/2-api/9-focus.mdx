## Focus  

`Focus` is an interface that describes single focus position and it's own css class inside grid

| Property name  |  Property description  |
|-----------    |  -------------------------|
| `rowId` | [Id](link) of row  |
| `columnId` | [Id](link) of column  |
| `className` | name of css class  |
