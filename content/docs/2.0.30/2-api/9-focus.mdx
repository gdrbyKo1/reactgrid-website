`Focus` is an interface that describes single focus position and it's own css class inside grid.

| Property name | Type          | Property description  |
| -----------   | -----         | --------- |
| `rowId`       | [`Id`](link)  | Row of cell to focus  |
| `columnId`    | [`Id`](link)  | Column of focused cell  |
| `className`   | `string`      | Name of CSS class  |
