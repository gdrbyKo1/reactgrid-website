`CellId` interface is used to describe position of cell (typically focused) inside grid

| Property name   | Type            | Property description  |
| -----------     | ----------      | -------------------------|
| `rowId`         | [`Id`](link)    | Id of row  |
| `columnId`      | [`Id`](link)    | Id of column  |
