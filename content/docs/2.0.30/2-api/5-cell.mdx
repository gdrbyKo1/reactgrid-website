`Cell` interface has only one necessary prop - `type`.

## Definition

```tsx
interface Cell {
    type: string;
    style?: CellStyle;
    className?: string;
}
```

## Properties

| Properties     | Type                 | Properties description |
| -----------    | ---------            | ---------- |
| `type`         | `string`             | Name of cell type |
| `style?`       | [`CellStyle`](link)  | Allowed style properties contained in [`CellStyle`](link) interface |
| `className?`   | [`string`](link)     | Additional css classes |
