## Cell  

`Cell` interface has only one necessary prop - `type`.

| Properties     | Type                  | Properties description  |
| -----------    | ---------             |  ---------- |
| `type`         | `string`              | Name of cell type |
| `style?`       | [`CellStyle`](link)   | Allowed styled contained in [`CellStyle`](link) interface |
