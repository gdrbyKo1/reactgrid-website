`CellTemplate` is used for the communication between ReactGrid and a cell.


| Name             | Type | Description |
|-----------       | ---- | -------------|
| `validate`       | `(cell: ` [`TCell`](link)`): ` [`CompatibleCell`](link)`<`[`TCell`](link)`>`| Validates passed data to cell based on type. If passed data are incorrect, it converts it to the target type and return cell with correct data. |
| `isFocusable?`   | `(cell: ` [`TCell`](link)`): boolean;` | Return `false` if you want that your cell shouldn't be focusable. |
| `update?`        | `(cell: ` [`TCell`](link)`, newCell: ` [`TCell`](link) ` `\|` ` [`CompatibleCell`](link)`: ` [`TCell`](link)`;`| Updates current cell. If not implemented, cell will be read-only. |
| `handleKeyDown?` | `TODO` | Returns the cell data either affected by the event or not. |
| `getStyle?`      | `(cell: ` [`TCell`](link)`, isInEditMode: boolean): ` [`CellStyle`](link)`;`| Return styles of cell. |
| `render`         | `(cell: ` [`TCell`](link)`, isInEditMode: boolean, onCellChanged: (cell: ` [`TCell`](link)`, commit: boolean) => void): React.ReactNode;`| Renders cell content.  |
