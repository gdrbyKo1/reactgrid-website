## Menu option  

Context menu is part of grid, each menu option must implement interface below:


| Property name     |  Property description |
|-----------    |  -------------------------|
| `id`        | text that identifies each menu option
| `label`      | label of menu option     |
| `handler`      | function that is called when option is clicked     |
