Row interface contains two necessary properties: `rowId` and `cells`.
It's essential to create any grid. 

## Definition

```tsx
interface Row {
    readonly rowId: Id;
    readonly cells: Cell[];
    // default: 25
    readonly height?: number;
    // default: false
    readonly reorderable?: boolean;
}
```

## Properties

| Property name     | Type                      | Property description  | 
| ------------      | -------------             | -------------         |
| `rowId`           | [`Id`](link)              | Unique id in all rows array | 
| `cells`           | [`Cell`](link)`[]`        | Array of `Cell` objects | 
| `height?`         | `number`, default: 25     | Height of each grid row (defaultly set to 25px)  | 
| `reorderable?`    | `boolean`, default: false | Property that allow row to change it's position in grid, default: `false` (row reorder implementation is on the developer side) |
