## Freezing panes

When you import ReactGrid with cellMatrixProps additional you have to copy below object to the existing columns and rows

```tsx
const frozenPanes = {
	frozenBottomRows: 1, // number of bottom frozen rows
	frozenLeftColumns: 1, // number of left frozen columns
	frozenRightColumns: 1, // number of right frozen columns
	frozenTopRows: 1 // number of top frozen rows
}
```

then append the `frozenPanes` object to `cellMatrixProps` using spread operator like below:

```tsx
<ReactGrid
    cellMatrixProps={{columns: getColumns(), rows: getRows(), ...frozenPanes}
/>
```
