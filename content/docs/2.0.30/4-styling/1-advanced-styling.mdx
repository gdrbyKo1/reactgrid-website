Reactgrid let you to style all key components using Sass variables. 

To override default styles of Reactgrid you have to assign new values ([available variables](link)) and then import all necessary `.scss` after.
If variable is not passed before imported `core.scss` it will by not modified.

Here's an example how to add custom focus style on your grid will have default value.

```scss
// Your custom color
$primary-color: #3579f8;

@import '/node_modules/@silevis/reactgrid/dist/lib/assets/core.scss';
```

If you use multiple grids on the same page, make sure your grid styles are 'local'.

```scss
// Independent Reactgrid classess
.reactgrid-red {
    $primary-color: red;
    @import '/node_modules/@silevis/reactgrid/dist/lib/assets/core.scss';
}

.reactgrid-blue {
    $primary-color: blue;
    @import '/node_modules/@silevis/reactgrid/dist/lib/assets/core.scss';
}

```

Wrap Reactgrid component into `div` element with class name:

```tsx

<div className="reactgrid-red">
    <Reactgrid {...props}/>
</div>

<div className="reactgrid-blue">
    <Reactgrid {...props}/>
</div>

```
