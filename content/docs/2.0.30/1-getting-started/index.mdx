#### Prerequisites
The only requirement in order to make ReactGrid work is an installed ReactJS library.

## 1. Add ReactGrid to your project

```shell
npm i @silevis/reactgrid
```

## 2. Import ReactGrid 

```tsx
import { ReactGrid } from '@silevis/reactgrid'
```

## 3. Create a CellMatrix

A CellMatrix is the data source for ReactGrid. It contains all information about all rows, columns and cells.

```tsx
const PlayerTable = (props) => {

  const cellMatrix = {
    columns: [
      { id: 'player', reorderable: true, resizable: true, width: 200 },
      { id: 'age',    reorderable: true, resizable: true, width: 50 },
      { id: 'club',   reorderable: true, resizable: true, width: 200 },
    ],
    rows: [
      { 
        id: 'header', 
        height: 25, 
        reorderable: true, 
        cells: [
          { type: 'header', data: 'Player' },
          { type: 'header', data: 'Age'},
          { type: 'header', data: 'Club' },
        ]
      },
      { 
        id: '1', 
        height: 25, 
        reorderable: true,  
        cells: [
          { type: 'text', data: 'Lionel Messi' },
          { type: 'number', data: 32},
          { type: 'text', data: 'Barcelona' },
        ]
      },
    ]
  }

  return <ReactGrid cellMatrixProps={cellMatrix} license='non-commerical'> 
}
```
